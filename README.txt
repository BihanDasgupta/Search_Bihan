🎀ྀིྀིInformation🎀ྀིྀི:

Too lazy to read through my profile? 
This is a website/portfolio I created in which you can search things about me. 
An LLM will answer your questions as a search result.

A default question is already put in place in the search bar: "Who is Bihan?" 
Upon opening the website, the search engine will search for the answer and display the search result.
To enter a different question, simply click on the search bar, erase the demo entry, and type away.
Your search result will be displayed below shortly after.

⟡ Note: The LLM can only answer with the information I feed it, mainly professional and work-related questions.
Knowledge on personal information is of course protected and limited. 
The time it takes for the search engine to display results varies on the complexity of the question you asked
and how long it takes the LLM to respond, so please be patient with her! 
If the website is down, it likely means I am rebooting it to make updates go through. 
It should be back up within 24 hours. ⟡
